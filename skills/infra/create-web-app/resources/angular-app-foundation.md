# Angular App Foundation

Copy files from `references/templates/app/` to `src/ui/src/app/`, replacing the Angular CLI generated defaults.

## Files Included

- `app.component.ts` - Root component with navbar visibility based on route data
- `app.component.html` - Conditional navbar + router-outlet
- `app.component.scss` - Full-height flex layout
- `app.config.ts` - Application configuration with interceptor, router, animations, markdown
- `app.routes.ts` - Empty routes array with example comments
